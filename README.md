# Hot & Cold 🔥🧊

A tiny desktop guessing game where the app tells you if you’re getting warmer or colder.  
Simple, fast, and mildly judgmental.

## What it is 🎯

Pick a level (`Easy`, `Medium`, `Epic`), guess a number, and follow the hot/cold hints until you win.

## Features ✨

- Clean desktop UI (CustomTkinter)
- 3 difficulty levels (1-10, 1-100, 1-1000)
- Color feedback based on how close your guess is
- Attempts counter + results screen
- Windows `.exe` build support via PyInstaller

## Run it 🚀

Open the Hot&Cold.exe file in the dist folder and enjoy!
